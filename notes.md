# DB 1 Notes

Database !== DBMS
Reletional DBMS (RDBMS) is a type of DBMS.

Examples of DBMS

- Postgres
- MySQL
- MongoDB (not relational)

Database Core Components

- Tables (relations)
  - rows (records)
  - columns (fields)
- Store Procedures/functions
- views
- indexes
